BioE131 Final Project

Our project explores how the Influenza A virus evolves in white-tailed eagles across different regions and years, specifically from 2018 to 2023. We're analyzing 4 key proteins, PB1, HA, NA, and PB2, to understand the virus's mutations and adaptations in birds. Using metrics such as mutation rates, conservation scores, and similarity scores, we aim to track how these changes impact the need for a new vaccine every year.

We've created an easy to use JBrowse2 database that you can explore. Click here for detailed instructions.

Ultimately, this work will help us better understand avian influenza and improve preparedness for future outbreaks.
